# Challenge 3 - Claim Notification Email Templates

Open `index.html` to preview six responsive claim lifecycle email templates. Templates use `{{variable}}` interpolation and sample data in `app.js`.

## Run

Open `index.html` directly or deploy the folder to a static host.

## Submission

- Live URL: https://19anim.github.io/ai-engineering-challenges/Challenge_3/
- Include `index.html`, `styles.css`, and `app.js`.

## Approach

- One template per claim lifecycle event.
- Shared interpolation helper replaces `{{variable}}` placeholders with sample data.
- Single-column email layout keeps the preview close to common email-client constraints.
- Rejection and approval states use visual emphasis for next steps and approved amount.
