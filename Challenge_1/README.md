# Challenge 1 - Insurance Plan Comparison Page

Static responsive comparison page for Bronze, Silver, and Gold insurance plans.

## Run

Open `index.html` directly or deploy the folder to any static host.

## Submission

- Include `index.html`, `styles.css`, `app.js`, and `data.js`.
- Live URL: https://19anim.github.io/ai-engineering-challenges/Challenge_1/

## Approach

- Renders all required plan data from `data.js`.
- Highlights strongest row values and shows unavailable benefits clearly.
- Uses a recommended badge based on value-for-money.
- Stacks into mobile-friendly cards on narrow screens.
